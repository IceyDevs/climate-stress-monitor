# Border Infrastructure Climate Stress Analyzer

A real-time environmental monitoring system that evaluates climate stress conditions in strategic border regions and estimates potential infrastructure risks.

## Features

• Climate Stress Index computation  
• Environmental driver analysis  
• Infrastructure risk assessment  
• Border town quick-selection  
• Interactive climate trend visualization  

## Technology Stack

Python  
Streamlit  
Plotly  
Open-Meteo API  

## Deployment

The system runs serverlessly using Streamlit Community Cloud.

## Data Source

Open-Meteo Weather API
